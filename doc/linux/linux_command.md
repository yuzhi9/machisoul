# Linux command

## Progress

```
$ ps [-ef] [-al]
$ ps -ef|grep [progress name]
```

## Compression

```
$ tar -[z]cvf <package name> <files/folders ...>
& tar -[z]xvf <package name>
```

## Permission

```
$ chmod [+] [1(x),2(w),4(r)] [-R] [files/folders ...]
$ chown <user> [files/folders ...]
```
